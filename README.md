# 📊 자산 배분 계산기 (Asset Allocation Calculator)

> 전체 투자 자산과 자산군별 비율 또는 금액을 설정하여 포트폴리오의 자산 배분 금액을 자동으로 계산해 주는 웹 도구입니다. 서버 없이 동작하는 클라이언트 사이드 단일 웹 앱(Single Web App)으로 구현되었습니다.

👉 **GitHub 저장소**: [https://github.com/juanlee01/Asset-Allocation-Calculator](https://github.com/juanlee01/Asset-Allocation-Calculator)

---

## ✨ 핵심 기능

1. **혼합 입력 방식 지원 (비율 % vs 금액 원)**
   - 각 자산 항목마다 **세그먼트 토글 버튼(비율/금액)**을 눌러 입력 방식을 직관적으로 바꿀 수 있습니다.
   - **비율(%) 모드**: 전체 자산 대비 설정한 비율만큼 투자 금액을 계산합니다. (소수점 입력 가능)
   - **금액(원) 모드**: 고정된 현금 금액을 직접 자산으로 입력하고, 전체 자산 대비 환산된 실제 비율을 자동으로 구해줍니다.

2. **실시간 자동 계산 및 콤마 포맷팅 UX**
   - 값을 입력하면 엔터 입력이나 새로고침 없이 하단 결과 테이블에 즉시 계산 결과가 반영됩니다.
   - 포커스 상태(Focus)에서는 천 단위 콤마가 제거되어 간편하게 수정할 수 있고, 포커스를 잃으면(Blur) 자동으로 콤마(`,`) 기호가 붙어 가독성이 높아집니다. **타이핑 도중 커서가 맨 뒤로 튀는 현상을 원천 방지**합니다.

3. **포트폴리오 총 비율 실시간 검증**
   - 설정한 자산의 총비율 합계가 `100.0%`에 부합하는지 소수점 오차범위(`0.01%`) 내에서 실시간으로 검증합니다.
   - 검증 상태에 따라 상단 바의 색상과 상태 텍스트가 바뀝니다:
     - **정상 (100.0%)**: <span style="color:#2e7d32">**녹색**</span>
     - **비율 부족 (<100.0%)**: <span style="color:#e65100">**주황색**</span> (부족한 비율 계산 표시)
     - **비율 초과 (>100.0%)**: <span style="color:#c62828">**빨간색**</span> (초과한 비율 계산 표시)
     - **계산 불가 (자산 0원)**: 자산 금액이 0원일 때 전용 상태 메세지 노출

4. **로컬 브라우저 자동 저장 (LocalStorage)**
   - 전체 투자 자산, 자산 이름, 모드(비율/금액), 설정값 등 모든 변경 사항이 브라우저 로컬 저장소에 실시간으로 저장됩니다.
   - 브라우저 창을 닫거나 새로고침해도 이전에 설정한 데이터가 자동으로 복원됩니다.

5. **다크 모드(Dark Mode) 지원**
   - 시스템 OS의 선호 테마를 기본으로 자동 반영하며, 헤더 우측의 테마 전환 버튼을 클릭하여 수동으로 다크/라이트 모드를 제어할 수 있습니다. 설정된 테마 정보도 브라우저에 저장되어 유지됩니다.

6. **웹 접근성(Accessibility) 및 모바일 최적화**
   - 모든 인풋과 버튼에 명확한 Label, `aria-*` 속성 및 `role`을 부여하여 스크린 리더와 키보드 탭 키로 완벽하게 조작 가능합니다.
   - 모바일 기기에서의 텍스트 입력을 위해 `inputmode="numeric"`을 적용하여 최적화된 숫자 키패드가 나타납니다.
   - 화면 해상도에 부드럽게 반응하는 반응형 그리드 시스템(모바일 100%, 데스크탑 최대 800px)이 설계되어 있습니다.

## 📂 파일 구조

서버나 빌드 도구 없이 브라우저에서 독립적으로 실행할 수 있도록 경량 파일 3개로 구조화되었습니다.
```text
/
├── index.html     # 계산기 화면 구조 및 웹 접근성 시맨틱 레이아웃
├── style.css      # 다크 모드, HSL 색상 변수, Segmented Control 디자인 및 반응형 스타일
├── script.js      # UUID 상태 관리, 혼합 계산 알고리즘, 포커스 제어 및 로컬 스토리지 연동
└── README.md      # 프로젝트 설명 문서
```

## 🚀 실행 및 사용 방법

별도의 패키지 설치나 개발 서버 설정이 필요하지 않습니다.

1. 이 저장소를 로컬로 클론하거나 다운로드합니다.
   ```bash
   git clone https://github.com/juanlee01/Asset-Allocation-Calculator.git
   ```
2. 프로젝트 폴더 내부의 `index.html` 파일을 크롬, 사파리, 엣지 등의 웹 브라우저로 실행합니다.
3. **전체 투자 자산**을 입력하고, 자산 항목의 이름과 비율 또는 금액을 설정하여 배분 금액 결과를 확인하세요.
