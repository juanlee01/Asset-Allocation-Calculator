# Asset Allocation Calculator Rules

## 프로젝트 개요
- 프로젝트명: Asset Allocation Calculator
- 목적: 전체 자산과 각 자산의 비율(%)을 입력하면 자동으로 투자 금액을 계산하는 웹 계산기 제작
- 구현 방식: Vanilla HTML / CSS / JavaScript
- 서버 없이 동작 가능한 정적 웹앱으로 구현
- 모바일과 데스크탑 모두 사용 가능해야 함

---

# 핵심 기능

## 1. 전체 자산 입력
- 사용자는 전체 자산 금액을 입력할 수 있어야 한다.
- 숫자만 입력 가능해야 한다.
- 천 단위 콤마 표시 지원.
- 음수 입력 불가.
- 빈 값이면 계산 불가.

예시:
- 10,000,000 원
- 2500000

---

## 2. 자산 항목 추가
사용자는 여러 자산 항목을 추가할 수 있어야 한다.

각 항목은:
- 자산 이름
- 비율(%)

을 가진다.

예시:

| 자산 | 비율 |
|---|---|
| 미국 주식 | 50 |
| 채권 | 30 |
| 금 | 20 |

---

## 3. 자동 계산
입력값이 변경되면 즉시 계산한다.

계산 공식:

```text
투자금액 = 전체자산 × (비율 / 100)
```

예시:

```text
전체 자산: 10,000,000
미국 주식: 50%

결과:
5,000,000 원
```

---

## 4. 총 비율 검증
- 전체 비율 합이 100%인지 실시간 검증
- 100%가 아니면 경고 표시
- 계산 결과는 계속 보여주되 상태를 시각적으로 표시

예시:
- 80% → 부족 경고
- 120% → 초과 경고
- 100% → 정상

---

## 5. 자산 항목 삭제
- 사용자는 자산 항목을 삭제할 수 있어야 한다.
- 최소 1개 항목은 유지.

  - desktop: 800px

## 입력 UX
- 숫자 입력 시 포커스 유지
- 입력값 변경 시 페이지 리로드 금지
- 엔터 없이 즉시 반영
- 모바일 키패드 최적화

```html

```

---

## 결과 표시
각 자산별:
- 자산 이름
- 비율
- 계산된 금액

을 명확히 표시한다.

예시:

| 자산 | 비율 | 투자 금액 |
|---|---|---|
| 미국 주식 | 50% | 5,000,000 원 |

## 색상 규칙
- 정상(100%): 녹색
- 부족(<100%): 주황색
- 초과(>100%): 빨간색
